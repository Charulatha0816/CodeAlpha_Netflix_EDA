# CodeAlpha_Netflix_EDA
# Netflix Data Analysis

## Internship

CodeAlpha Data Analytics Internship

## Project Objective

The objective of this project is to analyze the Netflix dataset and discover useful insights using Exploratory Data Analysis (EDA).

## Tools Used

* Python
* Pandas
* Matplotlib
* Google Colab

## Dataset

Netflix Titles Dataset

## Analysis Performed

1. Data loading and cleaning
2. Missing value analysis
3. Movies vs TV Shows comparison
4. Top 10 content-producing countries
5. Content release year analysis

## Key Findings

* Netflix has more Movies than TV Shows.
* United States is the largest content producer.
* India is the second-largest content producer.
* Netflix content has increased significantly in recent years.
